## Usage Patterns

### Trigger Achievement on Session Complete

```typescript
// In session completion handler
if (sessionMilestone) {
  setAchievementToShow({
    name: "First Steps",
    description: "Complete your first recitation session",
    icon: "mic",
    rarity: "common",
    xpReward: 25,
  });
}
```

### Show XP Award

```typescript
// After XP is awarded
setShowXPToast({
  amount: 50,
  multiplier: userStreak >= 7 ? 1.5 : 1,
  source: "Session Complete",
});
```

### Display User Progress

```typescript
// In header or dashboard
<XPProgress
  totalXP={user.totalXP}
  level={calculateLevel(user.totalXP)}
  currentLevelXP={user.totalXP % xpPerLevel}
  nextLevelXP={xpPerLevel}
/>
```
